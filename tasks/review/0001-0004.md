## Cases

### 1. `catch` collapsed to a one-liner in `configure-apps.nu`

Location: [configure-apps.nu](/Users/jem/dojo/nuparu/fixtures/dotfiles/scripts/configure-apps.nu:75)

What it is:

- `catch {|err| ... }` stayed compact and the outer closing brace stayed on the same line too:
  - `} catch {|err| print --stderr $err.msg } }`

Verdict:

- Do not like.

Why:

- It is technically understandable, but the doubled `} }` ending is cramped and easy to misread.
- This is not a multistatement-closure bug anymore; it is a simple-closure compaction style issue.

Likely cause:

- Simple closure compaction plus closure-closing-brace rejoining.
